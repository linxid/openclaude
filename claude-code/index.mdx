## 什么是 Claude Code？

Claude Code 是 Anthropic 推出的代理编码工具，它位于您的终端中，帮助您比以往任何时候都更快地将想法转化为代码。

## 核心功能

- **终端集成** - 直接在命令行中与 Claude 对话
- **IDE 集成** - 支持 VS Code、JetBrains 等主流 IDE
- **MCP 协议** - 通过 Model Context Protocol 扩展功能
- **Skills 系统** - 自定义技能扩展 Claude 的能力
- **Hooks 机制** - 自动化工作流程
