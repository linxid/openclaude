## 系统要求

### macOS
* macOS 13 Ventura 或更高版本
* Apple Silicon 或 Intel 处理器

### Linux
* 主流发行版（Ubuntu、Debian、Fedora 等）
* Node.js 22+

### Windows
* Windows 10/11 + WSL2
* 或 PowerShell 7+

## 推荐配置

| 组件 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 内存 | 4GB | 8GB+ |
| 存储 | 2GB | 10GB+ |
| 网络 | 宽带连接 | 稳定连接 |
